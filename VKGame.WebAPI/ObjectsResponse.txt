ВОЗРАЩАЕМЫЕ ОБЪЕКТЫ

Createbattle (localhost:5789/publicAPI/battle/start)
- Возращается при создании боя.
Поля: 
Status - статус создания битвы. Доступные значения: 1 - клан нашёл себе клана противника, 2 - клан стоит в очереди.
Id - Cодержит id клана с которым война, при status=2  возращается null, 
Name - Содержит имя клана с которым война, при status=2 - возращается null,
