
===========================================
ВОЗРАЩАЕМЫЕ ОБЪЕКТЫ

Createbattle (localhost:5789/publicAPI/battle/start)
- Возращается при создании боя.
Поля: 
Status(Integer) - статус создания битвы. Доступные значения: 1 - клан нашёл себе клана противника, 2 - клан стоит в очереди.
Id(string) - Cодержит id клана с которым война, при status=2  возращается null, 
Name(string) - Содержит имя клана с которым война, при status=2 - возращается null, 


UnitsList (localhost:5789/publicAPI/user/units)
 - Возращается при получении юнитов пользователя.
 Поля:
 Army(массив объектов Unit) - Массив объектов юнитов.

 
 Unit (localhost:5789/publicAPI/user/unitFromType)
 - Объект представляющий определённого юнита.
 Поля:
 Name(string) - имя юнита.
 Type(Integer) - тип юнита. Доступные значения: 1 - солдат, ...
 Damage(Integer) - Урон, наносимый юнитом.
 isOpen(Boolean) - Значение указывающее, открыт ли этот юнит у пользователя.
 
  
 Count(localhost:5789/publicAPI/army/count)
 - Возращается при получении количества сделанных юнитов.
 Поля:
 List(Массив объектов SpecificCount) - Массив объектов количества юнитов.
 
 
 SpecificCount(localhost:5789/publicAPI/army/countFromType)
 -Содержит количество готовой армии для определённого типа армии
 Поля:
 Id(Integer) - Тип армии
 Count(Integer) - Количество
 
 
 ChoiseMembers
 - Возращается при получении игроков для выбора.
 Поля: 
 List (Массив объектов ChoiseMember) - Массив объектов выбора пользователей
 
 ChoiseMember
 - Содержит определённый пользовательский выбор
 Поля:
 Number (Integer) - Номер топ.
 User (объект User) - Сам пользователь, информация о прокачке и т.д.
 
Attack 
- Ворзращается при атаке пользователя на пользователя.
Поля:
Status - Содержит статус выполнения атаки(см. ниже)
Damage - Количество нанесённого урона.
HpEnemy - Количество хп, отставшеся у пользователя.
IdBattack - Индентификатор казармы, которой нанесли урон.
Money - Количество валюты, которое пользователь получил за бой.
Доступные status-коды:
1 - Пользователь нанёс урон пользователю. Возращаются поля: Damage, HpEnemy
2 - Пользователь промазал. Возращаются поля: 
3 - Пользователь отнял казарму у пользователя. Возращаемые поля: IdBattack
4 - Пользователь нанёс урон и убил пользователя. Возращаемые поля: Damage, Damage, Money.
